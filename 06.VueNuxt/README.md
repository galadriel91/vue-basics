#  Vue2 + Nuxt

Vue FrameWork, Nuxt을 써보았다.

CLI와는 다르게 Vuex, Router가 설정되어 있는 점이 편리했다.

Layouts을 이용하여 공통부분을 정의 하는 점이 편했다.

앞으로 Nuxt을 사용해 더 많은 기능을 학습해야겠다.

추후 TypeScript 학습 후 Nuxt3을 써봐야지.
